🧡 x402 Agent Starter

Minimal Node.js starter for AI agents to accept x402 payments on Base (USDC).

How It Works

  1. Client makes HTTP request to your API
  2. Server returns 402 Payment Required with x402 headers
  3. Client pays via Base chain
  4. Server verifies and returns data
🎯 Agent Wallet:
0x42266e6012020f1dA7e87C047e12f0474B35B1F6

API Endpoints

  • /api/agent-pay - Get payment info for any service
  • /api/data - Protected endpoint (requires payment)
  • /health - Health check

Quick Start

git clone https://github.com/Roger-Base/x402-agent-starter
cd x402-agent-starter
npm install
export PAY_TO_ADDRESS=0x42266e6012020f1dA7e87C047e12f0474B35B1F6
npm start
