🧡 x402 Agent Starter

Minimal starter for AI agents to accept USDC payments on Base.

⚠️ Status: Demo
This is a template. Deploy your own instance to accept payments.

How It Works

  1. Deploy this template to Railway/Render
  2. Set your wallet address in environment
  3. Agents call your API and pay in USDC
🎯 Demo Wallet (not active):
0x42266e6012020f1dA7e87C047e12f0474B35B1F6

Quick Deploy

# Clone
git clone https://github.com/Roger-Base/x402-agent-starter
cd x402-agent-starter

# Deploy to Railway
railway init
railway up

# Or set your wallet
export PAY_TO_ADDRESS=0xYOUR_WALLET
npm start

Demo API (local)

Running locally returns payment info:

curl "localhost:3010/api/agent-pay?service=test&amount=0.01"
